# Specification

## Summary
**Goal:** Add comprehensive settings editors for videos and courses, and implement personalized course recommendations based on user activity.

**Planned changes:**
- Create video settings editor allowing owners to edit title, description, category, privacy, thumbnail, and tier restrictions
- Create course settings editor allowing creators to edit title, description, image, visibility, pricing, and tier requirements
- Add "Curated For You" course recommendations section on main page
- Implement backend recommendation algorithm using watch history, likes, comments, and followed channels

**User-visible outcome:** Video and course creators can edit all properties of their content after upload. Users see personalized course recommendations on the main page based on their viewing activity and channel follows.
